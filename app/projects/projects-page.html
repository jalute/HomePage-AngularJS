  • Video Poker Trainer

    The Video Poker Trainer was originally created as a Windows Phone application that has been converted over from C# to Javascript to be usable on this website (still in progress). The training application is to provide video poker basics, hold card tactics, and a optimal play simulator for the casino game Video Poker.

    Skills utilized:

    • Conception, research, and analysis to build this application from scratch.
    • Developed application in a very immature development environment (bleeding edge) and overcame numerous challenges with no guidance or reliance on developer forums.
    • Developed highly optimized engines to shuffle cards, statistical probabilities of hold cards, and rules engine to determine best hand to hold.
    • Follow strict Microsoft mobile guidelines (look and feel) for publishing to Microsoft Store.
    • Written in VS2012 - C# with XAML.
